# SIOP DID Profile


## Description

The spec is based on the SIOP specification that is part of the OIDC core specification.
The current working draft can be found here: https://identity.foundation/did-siop

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
